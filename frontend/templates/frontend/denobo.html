Denobo

Put together a multi-agent system at lightning speed, connecting agents in a network on the local machine, across VMs or even across network sockets and the internet. Just spin up a couple of agents: 

Agent myFirstAgent = new Agent("agent1");
Agent mySecondAgent = new Agent("agent2");
myFirstAgent.connectAgent(mySecondAgent);

Throw a handler on one of them: 

myFirstAgent.addMessageHandler(new MessageHandler() {

	@Override
	public void messageRecieved(Agent agent, Message message) {
		System.out.println("We recieved: '" + message.getData() + "' from '" +
			message.getOriginator() + "'!");
	}

});

And send messages between them containing any data you like until your heart's content, referring to your agents by nothing but their name: 

mySecondAgent.sendMessage("agent1", "Hello World!");

Get it!

Because this was originally designed for my second-year university project, you can find all the information you need in the obscene amount of JavaDoc in the source code and the accompanying quickstart guide.
