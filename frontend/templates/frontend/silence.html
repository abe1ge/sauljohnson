Silence

Recently, I had a database with a few hundred usernames that I needed to migrate to a new server. Sounds simple enough, right? Well, needless to say there was quite a substantial catch...

The only access I had to this data was through a webpage that was marked up inconsistently (making text-based parsing a chore) and I needed to get something together quite quickly. My first thought was a keyboard and mouse recorder that would record me copying one set of user details from the webpage into the database and then replay that action a few hundred times overnight. So out I went in search of a free tool I could use to accomplish this.

When I found nothing suitable or compatible that I didn't need to pay for, I opened up Visual Studio and put together Silence, a clean and simple keyboard and mouse recorder written in C# that lets your PC perform mouse and keyboard actions without you at the desk. In silence.

Get it!

This is easily one of the most useful programs I've ever put together, and although it isn't finished yet, it is up on my GitHub if you feel like downloading and building it to automate your simpler repetitive computing tasks. Heck, if you end up adding a useful feature or something then submit a pull request!
